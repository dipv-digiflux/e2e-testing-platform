üîó n8n Webhook Integration

This platform integrates with n8n workflows through webhooks:

  • Create User: POST https://dipv-digiflux-4.app.n8n.cloud/webhook/create-user
  • Create Project: POST https://dipv-digiflux-4.app.n8n.cloud/webhook/create-project
  • Create Test Case: POST https://dipv-digiflux-4.app.n8n.cloud/webhook/create-test-case
  • Run Tests: POST /webhook-test/run-tests (local) + POST https://dipv-digiflux-4.app.n8n.cloud/webhook/run-tests (n8n)

All forms automatically submit to these endpoints with proper error handling and validation.
